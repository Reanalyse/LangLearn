Data Flow

How data moves through the app from video load to subtitle display

USER opens video file via file dialog
↓ file path
MAIN PROCESS spawns:
  uv run python whisper_worker.py <video_path>
↓ stdout (SRT text)
WHISPER WORKER (Python)
  • Runs whisper with task="translate"
  • Japanese audio → English text + timestamps
  • Prints SRT to stdout
↓ parsed subtitle array via IPC
RENDERER stores subtitles in memory, shows progress bar during processing
  • Video is ready to play once processing completes
↓ user watches video...
USER clicks TRANSLATE button
  • video.pause()
  • find all subtitles where start >= currentTime - 5s and end <= currentTime
  • display overlay with matched lines
↓ user presses play
VIDEO resumes, overlay hides
